# Atividade Final - Introdução a Prática Experimental

## Começando

Essas instruções fornecerão uma cópia do projeto em funcionamento em sua máquina local para fins de desenvolvimento e teste.

## Pré-requisitos

OpenCV

Clone o repositório

## Instalando

<ul>
  
  <li>mkdir build && cd build</li>
  <li>cmake ..</li>
  <li>make</li>
  
</ul>
